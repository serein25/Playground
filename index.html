HTML & CSS

HTML

Document Structure

  • <html> - Root element of an HTML document.
  • <head> - Contains metadata and links.
  • <body> - Contains visible content.

Head Section

  • <title> - Sets the page title.
  • <meta> - Defines metadata.
  • <link> - Links external CSS.
  • <style> - Adds internal CSS.
  • <script> - Embeds JavaScript.

Text & Formatting

  • <h1> - <h6> - Headings from largest to smallest.
  • <p> - Defines a paragraph.
  • <span> - Inline container for styling.
  • <div> - Block container for grouping.

Lists

  • <ul> - Creates an unordered list.
  • <ol> - Creates an ordered list.
  • <li> - Defines a list item.

Links & Media

  • <a> - Creates a hyperlink.
  • <img> - Displays an image.
  • <video> - Embeds a video.
  • <audio> - Embeds audio.
  • <iframe> - Embeds another webpage.

Tables

  • <table> - Defines a table.
  • <tr> - Table row.
  • <th> - Table header cell.
  • <td> - Table data cell.

Forms & Inputs

  • <form> - Defines a form.
  • <input> - Creates an input field.
  • <button> - Adds a button.
  • <textarea> - Multi-line text input.
  • <label> - Labels input elements.
  • <select> - Creates a dropdown menu.
  • <option> - Defines dropdown options.

Layout & Sections

  • <header> - Defines a header.
  • <footer> - Defines a footer.
  • <section> - Groups related content.
  • <article> - Self-contained content.
  • <nav> - Defines navigation.
  • <aside> - Sidebar content.

CSS

Selectors

  • * - Selects all elements.
  • element - Selects a specific element.
  • .class - Selects elements with a class.
  • #id - Selects an element by ID.
  • element1, element2 - Selects multiple elements.
  • element element - Selects nested elements.
  • element > element - Selects direct children.

Box Model

  • width - Sets the element's width.
  • height - Sets the element's height.
  • margin - Space outside the element.
  • padding - Space inside the element.
  • border - Defines a border.
  • box-sizing - Controls box model behavior.

Text & Fonts

  • color - Sets text color.
  • font-size - Sets text size.
  • font-family - Defines font type.
  • font-weight - Adjusts text boldness.
  • text-align - Aligns text horizontally.
  • text-decoration - Adds/Removes underline, etc.
  • line-height - Adjusts line spacing.

Background & Colors

  • background-color - Sets background color.
  • background-image - Sets a background image.
  • background-size - Adjusts background scaling.
  • background-position - Positions the background.
  • background-repeat - Controls background repetition.
  • opacity - Adjusts transparency.

Positioning

  • position - Defines element positioning.
  • top, right, bottom, left - Sets positioning offsets.
  • z-index - Defines stack order.
  • display - Controls element display.
  • visibility - Shows or hides elements.

Flexbox

  • display: flex - Enables flexbox.
  • justify-content - Aligns items horizontally.
  • align-items - Aligns items vertically.
  • flex-direction - Defines main axis direction.
  • flex-wrap - Wraps items if needed.

Grid

  • display: grid - Enables grid layout.
  • grid-template-columns - Defines column structure.
  • grid-template-rows - Defines row structure.
  • gap - Sets space between grid items.
  • align-content - Aligns grid within container.

Animations & Effects

  • transition - Smooth property changes.
  • animation - Defines keyframe animations.
  • transform - Rotates, scales, moves elements.
  • box-shadow - Adds shadow around elements.
  • filter - Applies visual effects.
